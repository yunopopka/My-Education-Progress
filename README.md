# My-Education-Progress
Это репозиторий поего последовательного обучения и развития, внутри вы найдете две разные папки с разным назначением.
SandboxEducation - Здесь хранятся файлы для самообучения, черновики, ежедневные упражнения и эксперименты с кодом. Структура может быть хаотичной, важен сам процесс практики.
ShowCase - Здесь собраны мои лучшие проекты, экзаменационные программы и примеры чистого кода, демонстрирующие уровень владения ООП и архитектурными паттернами.
